## 🎯 Funcionalidades

### 📈 Análises Disponíveis

- **📊 Absenteísmo**: Índices de ausência, análise de Bradford, tendências
- **📈 COPSOQ III**: Riscos psicossociais completos (26 dimensões)
- **🔥 Burnout (CBI)**: Esgotamento pessoal, relacionado ao trabalho e ao cliente
- **💼 Workaholism (DUWAS)**: Trabalho excessivo e compulsivo
- **🔄 Turnover**: Rotatividade de pessoal e impacto financeiro

### 🤖 Assistente IA

- Consultor de RH com memória persistente
- Análises automatizadas usando Google Gemini 2.0
- Recomendações personalizadas baseadas em análises
- Base de conhecimento integrada

### 📄 Recursos Adicionais

- ✅ Exportação para Excel e CSV
- ✅ Dashboard interativo com visualizações
- ✅ Sistema de backup e restauração
- ✅ Base de conhecimento com busca inteligente
- ✅ Histórico completo de análises
